# web-basic
## Tag
* bold: <strong>word</strong>
* underline: <u>word</u>
* headline: <h1>word</h1> ... <h6>word</h6>
* line break: <br>
* paragraph: <p>paragraph</p>
    + <p></p> tag can only make specified amount of margin 
        + CSS: <p style="margin-top:45px;">
* attribute
    + image tag: <img src="a.jpg" width="n%">
